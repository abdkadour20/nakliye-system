المرحلة 6 - Real-time Multi Device Sync

ما الذي أُضيف؟
- خدمة Supabase Realtime تستمع لتغييرات الجداول الأساسية.
- إذا أضاف جهاز آخر رحلة/سائق/مركبة/سند، يحاول البرنامج تحميل آخر البيانات تلقائياً.
- ما زالت المزامنة اليدوية موجودة كزر احتياطي.
- ما زال Offline يعمل عند انقطاع الإنترنت.

خطوات التشغيل:
1) افتح Supabase > SQL Editor.
2) شغّل الملف: db/phase6_realtime_sync.sql
3) شغّل البرنامج:
   npm install
   npm start
4) افتح البرنامج من جهازين أو نافذتين.
5) أضف رحلة من جهاز، وانتظر لحظات لترى التحديث في الجهاز الآخر.

ملاحظة أمان:
حالياً تم تعطيل RLS للجداول أثناء مرحلة التطوير لتسهيل الربط. بعد تثبيت نظام المستخدمين النهائي سنعيد تفعيل الحماية بسياسات صحيحة.
