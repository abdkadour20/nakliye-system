المرحلة 3 - PDF Engine Ready

تمت إضافة خدمة PDF احترافية اختيارية:

src/services/pdfEngine.js

الميزات:
- تصدير أي عنصر HTML إلى PDF حقيقي.
- دعم A4.
- دعم الصفحات المتعددة.
- لا يكسر البرنامج إذا لم تكن المكتبات مثبتة.
- يعطي رسالة واضحة عند غياب المكتبات.

لتفعيل PDF الحقيقي داخل المشروع:

npm install jspdf html2canvas

بعدها يمكن ربط الخدمة بأزرار الفاتورة والتقارير تدريجياً.

ملاحظة:
الفواتير الحالية تعمل بالطباعة الاحترافية كما هي، وهذه المرحلة تضيف محرك PDF جاهز للدمج بدون كسر النظام.
