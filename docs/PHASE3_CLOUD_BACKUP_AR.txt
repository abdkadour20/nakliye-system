المرحلة الثالثة: Cloud Backup تلقائي

ما تمت إضافته:
1) إنشاء نسخة Backup تلقائية يومية على Supabase بعد نجاح المزامنة.
2) عند الضغط على زر المزامنة اليدوية يتم إنشاء Backup يدوي أيضًا.
3) تم حفظ حالة آخر Backup داخل المتصفح لعرضها في صفحة Cloud.
4) إذا كان الإنترنت مقطوعًا، يبقى Backup مطلوبًا ويتم تنفيذه لاحقًا عند رجوع الاتصال.

خطوات التفعيل:
1) افتح Supabase.
2) اذهب إلى SQL Editor.
3) شغّل الملف:
   db/supabase_phase3_backup_migration.sql
4) أعد تشغيل البرنامج.
5) من صفحة Cloud اضغط: Manuel Senkronize Et.

ملاحظات مهمة:
- هذه المرحلة لا تحذف أي بيانات محلية.
- Backup يحتوي على الرحلات، السائقين، المركبات، السندات، المستخدمين، الوثائق، السجلات، ومعلومات الشركة.
- المرحلة القادمة ستكون لتقوية تسجيل الدخول والصلاحيات وربط المستخدمين بـ Supabase Auth.
