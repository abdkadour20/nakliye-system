V7 Phase 4 - PDF حقيقي

تم تفعيل أزرار PDF حقيقية داخل البرنامج:
- زر Fatura PDF لكل رحلة
- زر PDF للتقارير والفلاتر
- إنشاء PDF مباشر بواسطة jsPDF + html2canvas
- لا يعتمد على نافذة طباعة Chrome

مهم:
قبل التشغيل تأكد من تنفيذ:
npm install jspdf html2canvas

طريقة الاستخدام:
استبدل الملفات الجديدة ثم شغل npm start.
